# 개요
- 학습 목표
    - 해당 기술들이 왜 필요한지
    - 각 기술의 장단점이 무엇인지
- 다양한 데이터 접근 기술 방식
    - SQLMapper : 개발자가 SQL만 작성하면 그 결과를 객체로 편리하게 매핑
        - JdbcTemplate
        - MyBatis
    - ORM 관련 기술 : 저장하고 싶은 데이터를 자바 컬렉션에 저장하고 조회하듯이 사용 가능하게 함, SQL을 만들어줌
        - JPA, Hibernate
        - 스프링 데이터 JPA
        - Querydsl
